初学python

面向对象、解释型、动态数据类型的高级程序设计语言

作者：荷兰人Guido van Rossum于1989年发明，第一个公开发行版发行于1991年，遵循GPL协议

优点

  • 简单易学
  • 免费开源
  • 已于维护
  • 代码易读性高，编程方式较为优雅
  • 可移植性强
  • 面向对象
  • 可嵌入
  • 一个标准的
  • 支持GUI编程

缺点

  • 运行速度慢（与C语言相比，比起JAVA性能也较差）
  • 代码不可加密

python能做什么

  • 网络后端
  • 自动化运维
  • 数据分析
  • 自动化测试
  • 网络爬虫
  • 人工智能

python的三种运行方式

  • 交互式解释其模式
  • 命令行脚步模式
  • 集成开发环境

python的编码规范

  • 不以分好结尾
  • 尽量控制每行代码的长度，不要多于80个字符
  • 代码的包含关系用缩进控制，每一层级最好用四空格控制
  • 顶级定义之间空两行，方法之间空一行
  • 每个语句都应该占据一行
  • 不要滥用括号
  • 文件名不能出现中文或空格

标识符规则

  • 只能由字母、数字、下划线组成
  • 开头不能是数字
  • 不能是python的关键字
    例如：def False True and break class del 等等
  • 区分大小写
  • 取名字要做到见名知义
